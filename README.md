# Final Grade Calculator

## What this tool does
This is an interactive web tool that helps students calculate their current and projected grade in a course. Users can add assignments, enter scores and weights, and instantly see updated results.

## How to use it
1. Enter an assignment name
2. Enter points earned
3. Enter points possible
4. Enter the assignment weight
5. Click **Add Assignment**
6. View your updated grade summary
7. Delete rows or reset everything if needed
